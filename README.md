# GeoExpert — Образовательный тренажер по географии

## Описание
GeoExpert — это интерактивное веб-приложение на Django для изучения стран, столиц и флагов. 
Проект разработан в рамках учебного модуля.

### Основной функционал:
- **Обучение по регионам:** возможность выбора конкретного материка для изучения.
- **Тренажер (Квиз):** интерактивный режим проверки знаний с фиксацией ошибок.
- **База знаний:** полный список стран с возможностью редактирования заметок.
- **CRUD:** реализованы функции добавления, просмотра, изменения и удаления стран.
- **Валидация:** строгая проверка вводимых данных (защита от цифр в названиях).

## Технологии
- Python 3.11 / Django 5.2
- Bootstrap 5 / Animate.css
- Pylint (Оценка 8.04/10)

## Установка и запуск
1. Склонируйте репозиторий.
2. Создайте виртуальное окружение: `python -m venv venv`.
3. Активируйте его: `source venv/bin/activate`.
4. Установите зависимости: `pip install django django-bootstrap5 requests`.
5. Примените миграции: `python manage.py migrate`.
6. Наполните базу данными: `python fill_db.py`.
7. Запустите сервер: `python manage.py runserver`.
